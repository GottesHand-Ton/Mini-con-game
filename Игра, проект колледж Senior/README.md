# Подземный лабиринт - Текстовый квест

Игра-текстовый квест, в которой игрок управляет персонажем, пытающимся выбраться из подземного лабиринта.

## Как запустить игру

### Способ 1 (с Makefile)

make
./game.exe

### Способ 2 ручная компиляция
g++ main.cpp gg.cpp map.cpp -o game.exe
./game.exe

### Управление
Клавиша	Действие
W / A / S / D	Движение вверх, влево, вниз, вправо
I	Открыть инвентарь
P	Сохранить игру
Q	Выйти из игры

### Что нужно для запуска
Компьютер с Windows (используется библиотека windows.h)
Установленный компилятор g++ (MinGW) или любой другой C++ компилятор

### Правила игры
Персонаж начинает в первой локации (отмечена цифрой 1)
Нужно добраться до последней 5-й локации
Некоторые локации изначально закрыты, их нужно открыть, решив головоломку
За неправильное решение головоломки персонаж теряет здоровье
### На локациях можно найти:
Сундуки (дают зелья лечения)
NPC (с ними можно поговорить или сразиться)

### В инвентаре можно:
Использовать зелья для восстановления HP
Экипировать оружие для увеличения урона (доступно только одно оружие одновременно)
Боевая система пошаговая: можно атаковать, пить зелье или сбежать

### Игру можно сохранить (кнопка P) и загрузить при следующем запуске
Все события записываются в файл log.txt
